# Governance Proposal Types

## Overview

Tokamak DAO supports different types of governance proposals, each with specific requirements and procedures.

## Type A Proposals

### Scope
- Staking parameter modifications
- Inflation rate adjustments
- SeigManager configuration changes

### Requirements
- Higher quorum threshold
- Longer voting period

## Type B Proposals

### Scope
- Smart contract upgrades
- Protocol parameter changes
- Committee membership changes

### Requirements
- Standard quorum threshold
- Technical review required
