# Tokamak DAO Voting Rules

## Overview

This document describes the voting rules and procedures for Tokamak DAO governance.

## Quorum Requirements

- Minimum quorum: 5% of total staked TON
- Voting period: Minimum 3 days

## Voting Power

- Voting power is determined by staked TON amount
- 1 staked TON = 1 vote

## Proposal Types

### Type A: Staking and Inflation
- Changes to staking parameters
- Inflation rate adjustments
- Reward distribution modifications

### Type B: Contract Upgrades
- Smart contract upgrades
- Protocol parameter changes
- Security-related updates
