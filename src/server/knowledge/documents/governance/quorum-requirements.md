# Quorum Requirements

## Overview

Quorum is the minimum participation required for a vote to be valid.

## Current Quorum Settings

- **Minimum Quorum**: 5% of total staked TON must participate
- **Voting Period**: Minimum 3 days for all proposals

## Quorum Calculation

```
Required Votes = Total Staked TON Ã— 0.05
```

## Special Cases

- Emergency proposals may have different requirements
- Protocol upgrade proposals may require higher quorum
