# DAO Committee Contract

## Overview

The DAO Committee contract manages governance proposals and voting in Tokamak DAO.

## Contract Information

- **Network**: Ethereum Mainnet
- **Role**: Governance execution and proposal management

## Key Functions

### Create Proposal
Submit a new governance proposal for voting.

### Vote
Cast a vote on an active proposal.

### Execute Proposal
Execute a passed proposal after voting period ends.

## Committee Members

- Committee members have special privileges for certain actions
- Membership can be changed through governance proposals
